# Model Interpretability Datasets

This folder contains the **dataset** used in the **classification** and **regression models**.

- [**Boston.csv**](Boston.csv) is **used for regression** models. This dataset was taken from the [**UCI Machine Learning Repository**](https://archive.ics.uci.edu/ml/index.php) from this [**link**](https://archive.ics.uci.edu/ml/datasets/Housing).

- [**mushrooms.csv**](mushrooms.csv) is **used for classification** models. This dataset was taken from the [**UCI Machine Learning Repository**](https://archive.ics.uci.edu/ml/index.php) from this [**link**](https://archive.ics.uci.edu/ml/datasets/Mushroom).
